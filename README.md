## Overview

Jwrapper was designed to be a simple wrapper program that runs any Java application without having the user issue commands everytime.

## How to start?

	curl -L https://github.com/vintage-wang/jwrapper/zipball/master > a.zip
	unzip a.zip
	rm -f a.zip
